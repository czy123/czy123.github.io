ngui和unity5.2.2的bug

  • unity

Unity 5.2.1 + NGUI 3.9.2

经常 crash 在保存场景的时候

完美解决方案

修改ngui uilabel代码
line 1139 : #if !UNITY_4_3 && !UNITY_4_5 && !UNITY_4_6

修改成 : if !UNITY_4_3 && !UNITY_4_5 && !UNITY_4_6 && !UNITY_5

但是还有点问题是ngui使用动态文字，我感觉是ngui的问题，目前解决方案是把3.8.0的修改到3.9.2
